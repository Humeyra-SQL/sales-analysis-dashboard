\# 📊 Sales Performance Dashboard (Power BI)



\## 📌 Overview

This project presents a sales analytics dashboard built with Power BI.  

It provides insights into revenue, profit, and regional performance.



\## 🚀 Features

\- KPI tracking (Revenue, Profit, Orders)

\- Year-over-Year analysis

\- Sales by category

\- Regional sales map

\- Interactive filtering



\## 🛠️ Tools Used

\- Power BI

\- DAX

\- Data Modeling
